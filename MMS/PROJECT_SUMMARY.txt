QUICK START (5 MINUTES)
================================================================================

1. Prerequisites:
   - Node.js 18+
   - Java 17+
   - Maven 3.6+
   - PostgreSQL 12+

2. Database Setup:
   createdb mms_db
   (Run SQL from QUICKSTART.md)

3. Backend:
   cd backend
   mvn spring-boot:run
   (Runs at http://localhost:8080)

4. Frontend:
   npm install
   npm start
   (Runs at http://localhost:4200)

5. Login:
   Username: admin
   Password: admin
